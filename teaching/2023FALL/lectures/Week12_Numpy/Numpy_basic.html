Permutations (1d)

  • permutation:
    • Randomly permute a sequence, and return the result
rng = np.random.default_rng(seed=42)
arr1 = np.arange(6)
arr1
## array([0, 1, 2, 3, 4, 5])
rng.permutation(arr1)
## array([3, 2, 5, 4, 1, 0])
arr1 ## the original array remains the same
## array([0, 1, 2, 3, 4, 5])
  • shuffle:
    • in place operation, modify the original array
rng = np.random.default_rng(seed=42)
arr1 = np.arange(6)
rng.shuffle(arr1)
arr1
## array([3, 2, 5, 4, 1, 0])

Permutations (2d) permutation() method

  • permutation:
    • Randomly permute a sequence, and return the result
    • axis = 0: permute rows
    • axis = 1: permute columns (within each row)
rng = np.random.default_rng(seed=42)
arr1 = np.arange(12).reshape(4,3)
rng.permutation(arr1) ## default axis=0
## array([[ 9, 10, 11],
##        [ 6,  7,  8],
##        [ 3,  4,  5],
##        [ 0,  1,  2]])
rng.permutation(arr1, axis=1)
## array([[ 2,  0,  1],
##        [ 5,  3,  4],
##        [ 8,  6,  7],
##        [11,  9, 10]])

Permutations (2d) shuffle() method

  • shuffle:

    • axis = 0: permute rows
    • axis = 1: permute columns (within each row)

Unique and Other set logic

x = np.array([0,1,0,2,5,6,5,0])
np.unique(x)
## array([0, 1, 2, 5, 6])
y = np.array([0,1,2,3])
np.intersect1d(x,y)
## array([0, 1, 2])
np.union1d(x,y)
## array([0, 1, 2, 3, 5, 6])
np.in1d(x,y)
## array([ True,  True,  True,  True, False, False, False,  True])
np.setdiff1d(x,y)
## array([5, 6])
